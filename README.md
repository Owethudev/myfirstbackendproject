Live Demo

Frontend: https://snplport.netlify.app

Frontend

- React
- Vite
- TypeScript
- Tailwind CSS

Backend

- Node.js
- Express.js
- MongoDB

project-root/
├── backend/
└── frontend/

💻 Running the Project Locally

1. Clone the repository https://github.com/Owethudev/myfirstbackendproject.git

git clone

2. Start the Backend

Navigate to the backend directory:

cd backend

Install dependencies:

npm install

Start the development server:

npm run dev

3. Start the Frontend

Open a new terminal and navigate to the frontend project:

cd frontend
cd myfullstack

Install dependencies:

npm install

Start the Vite development server:

npm run dev

The frontend will typically be available at:

http://localhost:5173
